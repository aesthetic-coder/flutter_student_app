# Student Productivity App - Flutter

A modern, feature-rich Flutter application designed for university students to manage their academic life.

## Features

✅ **Semester Management** - Create, archive, and switch between semesters  
✅ **Subject/Course System** - Organize subjects with credits, instructors, and classroom info  
✅ **Assignment Tracker** - Track assignments with due dates and priorities  
✅ **Exam Manager** - Schedule exams and track study progress  
✅ **Attendance Tracker** - Mark attendance and monitor percentages  
✅ **GPA Calculator** - Calculate semester and cumulative GPA  
✅ **Material 3 Design** - Beautiful modern UI with smooth animations  
✅ **Dark Mode Support** - Seamless theme switching  
✅ **Offline-First** - All data stored locally with GetStorage  

## Getting Started

### Prerequisites
- Flutter 3.0+
- Dart 3.0+

### Installation

```bash
git clone https://github.com/aesthetic-coder/flutter_student_app.git
cd flutter_student_app
flutter pub get
flutter run
```
